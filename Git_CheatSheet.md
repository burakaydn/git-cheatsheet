* **Create new repository**
> git init

* **Configure git settings**
> git config –global user.name “GDG Ankara”
> git config --global user.email "eposta@dresi.com"

* **Add changes to Index**
> git add .

* **Repository status**
> git status

* **Commit changes**
> git commit -m "Why?"

* **Add remote repository**
> git remote add remote_repo_name remote_repo_path

* **Push changes to remote repository**
> git push remote_repo local_branch

* **Clone repository**
> git clone repository_path

* **Update local repository from remote**
> git pull

* **List branches**
> git branch

* **Create a new branch**
> git checkout -b branch_name

* **Switch between branches**
> git checkout branch_name

* **Delete branch**
> git branch -d branch_name

* **Merge branches**
> git merge otherbranch

* **Restore file fom last commit**
> git checkout -- File.java

* **View Changes**
> git diff

* **Reset to previous commit**
> git reset

* **Remove file from Index**
> git rm File.java

* **Remove untracked files**
> git clean -f

* **Remove ignored files**
> git clean -f -X
